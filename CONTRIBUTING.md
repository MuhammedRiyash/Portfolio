# Contributing to Muhammed Riyash's Portfolio

First off, thank you for considering contributing to this portfolio project! It's people like you who help improve this showcase.

## How Can I Contribute?

### Reporting Bugs
If you find a bug or a broken link, please create an issue or contact the admin. Before creating a bug report, please check that the issue hasn't already been reported.

### Suggesting Enhancements
If you have an idea for a new feature, a design improvement, or a better way to showcase projects, feel free to share it!
